## Descripción
Vamos a desarrollar una aplicación como proyecto final de Desarrollo de Aplicaciones Multiplataforma.

Dicho proyecto será una red social de fotografías, en la que los usuarios podrán subir fotografías asociadas a recetas, y compartirlas con sus seguidores.

Iremos subiendo los avances del desarrollo de código a este repositorio.
